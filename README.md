# 🏢 Active Directory Home Lab

## 📋 Overview

This project documents the deployment and configuration of an Active Directory environment using Windows Server 2022 and Windows 11. The lab was designed to simulate common IT Support and Help Desk tasks performed in enterprise environments, including user management, authentication, Group Policy, and domain administration.

---

## 🛠️ Technologies Used

- Windows Server 2022
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy
- Oracle VirtualBox
- Command Prompt

---

## 🎯 Objectives

- Deploy Active Directory Domain Services
- Promote a Windows Server to a Domain Controller
- Create and manage domain users
- Configure Organizational Units (OUs)
- Configure Security Groups
- Join a Windows workstation to the domain
- Verify domain authentication
- Configure password policies using Group Policy
